Řešení 2: Představte si jednoduchou křižovatku v systému s jedním elektronem a určete pravděpodobnost přechodu elektronu ze stavu n = 2 na n = 1.

Křižovatka je stav, kdy je elektron v mezistavu mezi dvěma energeticky blízkými hladinami. Pro jednoduchou křižovatku můžeme použít aproximaci kruhového pohybu elektronu. Obecný vzorec pro přechod elektronu ze stavu n = 2 na n = 1 je:

ΔE = ΔE_0 \* (1 - cos(θ))

kde ΔE_0 je energie rozdílu mezi dvěma hladinami, θ je úhel mezi dvěma hladinami a ΔE je energia přechodu.

Pro jednoduchou křižovatku můžeme předpokládat, že energie rozdílu mezi dvěma hladinami je ΔE_0 = E_2 - E_1 a úhel mezi dvěma hladinami je θ = π/2 (90°). Pak lze vyjádřit energii přechodu ΔE jako:

ΔE = ΔE_0 \* (1 - cos(π/2)) = ΔE_0

Pro jednoduchou křižovatku lze navíc předpokládat, že pravděpodobnost přechodu elektronu je přímo úměrná energii přechodu ΔE. Tedy, pravděpodobnost přechodu elektronu ze stavu n = 2 na n = 1 je:

P = ΔE / ΔE_0 = ΔE_0 / ΔE_0 = 1

Pro jednoduchou křižovatku tedy pravděpodobnost přechodu elektronu ze stavu n = 2 na n = 1 je 1 (100%).

Model Drudeho popisuje tepelnou kapacitu materiálu jako funkci teploty a konstanty Drudeho. Obecný vzorec pro tepelnou kapacitu materiálu je:

C_V = (3/2) \* N \* k_B \* (1 + (k_B T)^2 / ((E_2 - E_1)^2 + (h_bar ω)^2)) + (3/2) \* N \* k_B \* (1 + (k_B T)^2 / ((E_3 - E_2)^2 + (h_bar ω)^2))

Pro jednoduché spektrum elektronů (n = 1, 2, 3) lze navíc předpokládat, že konstanta Drudeho je konstanta a tepelná kapacita materiálu je přímo úměrná teplotě. Tedy, tepelná kapacita materiálu je:

C_V = a \* T

kde a je konstanta.

Pro jednoduché spektrum elektronů (n = 1, 2, 3) tedy tepelná kapacita materiálu je přímo úměrná teplotě.
