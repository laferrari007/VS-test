Řešení 1: Rozdíl mezi síťovým rozhraním Ethernet a Wi-Fi

Síťové rozhraní Ethernet a Wi-Fi jsou dva rozdílné typy sítí, které se používají pro připojení k počítačové síti. Hlavními rozdíly mezi nimi jsou:

1. **Typ přenosu dat**: Ethernet využívá kabelové připojení k síti, zatímco Wi-Fi používá bezdrátové připojení. Ethernet je rychlejší a spolehlivější, ale vyžaduje fyzickou přítomnost kabelu. Wi-Fi je flexibilnější a umožňuje přístup k síti z větší vzdálenosti, ale může být méně spolehlivý.
2. **Vzdálenost připojení**: Ethernet má omezenou vzdálenost připojení, obvykle do 100 metrů. Wi-Fi má mnohem větší vzdálenost připojení, obvykle do 100 metrů nebo více, ale může být ovlivněna fyzickým prostředím a přítomností jiných zařízení.
3. **Rychlost přenosu dat**: Ethernet je rychlejší než Wi-Fi. Ethernet může dosáhnout rychlostí až 10 Gb/s, zatímco Wi-Fi má rychlost obvykle do 1,9 Gb/s.

Řešení 2: Fungování protokolu DHCP

Protokol DHCP (Dynamic Host Configuration Protocol) je používaný pro přidělování IP adres a jiných síťových parametrů zařízením v počítačové síti. Proces získání IP adresy pro zařízení je následující:

1. **Zařízení se spojí do sítě**: Zařízení se připojí k síti pomocí Ethernetu nebo Wi-Fi.
2. **Zařízení odesílá požadavky**: Zařízení odesílá požadavky na DHCP server v síti.
3. **DHCP server přiděluje IP adresu**: DHCP server přidělí IP adresu a další síťové parametry zařízení.
4. **Zařízení přijme IP adresu**: Zařízení přijme IP adresu a další síťové parametry a nastaví své síťové rozhraní.
5. **Zařízení oznámí svůj přítomnost**: Zařízení oznámí svůj přítomnost v síti a aktualizuje své síťové parametry.

Řešení 3: Rozdíl mezi routerem a switchem v počítačové síti

Router a switch jsou dva typy zařízení, které se používají v počítačové síti. Hlavními rozdíly mezi nimi jsou:

1. **Funkční role**: Router je zařízení, které směruje datová balíčka z jedné sítě do jiné sítě. Switch je zařízení, které směruje datová balíčka mezi zařízeními v jedné síti.
2. **Připojení**: Router má několik sítí připojených k němu, zatímco switch má pouze jednu síť připojenou k němu.
3. **Rychlost přenosu dat**: Router je rychlejší než switch, protože musí směrovat datová balíčka mezi sítěmi.
4. **Použití**: Router se používá pro směrování dat mezi sítěmi, zatímco switch se používá pro směrování dat mezi zařízeními v jedné síti.

Řešení 4: Výhody a nevýhody používání statické IP adresy na zařízení v počítačové síti

Statická IP adresa je IP adresa, která je přidělena zařízení v síti na trvalou dobu. Výhody a nevýhody používání statické IP adresy jsou:

**Výhody:**

* **Stabilita**: Statická IP adresa je stabilní a nezávislá na jiných zařízení v síti.
* **Správa**: Statická IP adresa je snadno spravovatelná, protože je přidělena na trvalou dobu.
* **Bezpečnost**: Statická IP adresa je bezpečnější než dynamická IP adresa, protože je méně pravděpodobné, že bude změněna.

**Nevýhody:**

* **Omezenost**: Statická IP adresa je omezená na jednu síť a nemůže být použita v jiné síti.
* **Komplexita**: Statická IP adresa vyžaduje větší úsilí pro správu a konfiguraci.
* **Nedostupnost**: Statická IP adresa může být nedostupná v případě, že je přidělena jinému zařízení v síti.

Řešení 5: Fungování protokolu DNS

Protokol DNS (Domain Name System) je používaný pro převod doménového jména na IP adresu. Proces převodu doménového jména na IP adresu je následující:

1. **Používání doménového jména**: Uživatel použije doménové jméno pro přístup k webové stránce nebo jiným službám.
2. **Převod doménového jména na IP adresu**: DNS server převádí doménové jméno na IP adresu.
3. **Odesílání požadavku**: Uživatel odesílá požadavek na webovou stránku nebo jinou službu.
4. **Přijetí požadavku**: Webová stránka nebo jiná služba přijme požadavek a vrátí odpovídající datová balíčka.
5. **Předání dat**: Data jsou předána uživateli a zobrazí se na obrazovce.

Řešení 6: Bezpečnost počítačové sítě a význam používání firewallov, antivirusových programů a hesel

Bezpečnost počítačové sítě je zajištěna pomocí různých nástrojů a postupů. Význam používání firewallov, antivirusových programů a hesel je:

1. **Firewallov**: Firewallov chrání počítačovou síť před nelegálními aktivitami a malwarem.
2. **Antivirusových programů**: Antivirusových programů chrání počítačovou síť před malwarem a škodlivými programy.
3. **Hesel**: Hesel chrání přístup k počítačové síti a chrání před nelegálními aktivitami.

Řešení 7: Rozdíl mezi IPv4 a IPv6

IPv4 a IPv6 jsou dva typy IP adres, které se používají v počítačové síti. Hlavními rozdíly mezi nimi jsou:

1. **Délka**: IPv4 má délku 32 bitů, zatímco IPv6 má délku 128 bitů.
2. **Poměr**: IPv4 má poměr 1:65536 (1 IP adresa na 65 536 zařízení), zatímco IPv6 má poměr 1:340 282
