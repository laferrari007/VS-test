Řešení 1:
Vytvořte funkci `vypisTabulku`, která vypíše tabulku s názvy dní v týdnu.

```python
def vypisTabulku():
    # Definice dní v týdnu
    dny_v_tydeni = ["Pondělí", "Úterý", "Středa", "Čtvrtek", "Pátek", "Sobota", "Neděle"]
    
    # Vypis tabulky
    print("  ", end="")
    for den in dny_v_tydeni:
        print(f"{den[:3]:^8}", end="")
    print()
    print("-" * 38)
    for i in range(7):
        print(f"{i+1:^2} ", end="")
        for den in dny_v_tydeni:
            print(f"{den[i]:^8}", end="")
        print()

# Zvolení funkce
vypisTabulku()
```

Explanace: Tato funkce definuje seznam dní v týdnu a poté pomocí cyklu vypíše tabulku s názvy dní. Každý den je vypsaný na samostatné řádce, přičemž první řádek obsahuje názvy dní a druhý řádek obsahuje čísla od 1 do 7. Všechny názvy dní jsou středně na řádku.

Řešení 2:
Vytvořte program, který zobrazí počet dní v měsíci. Program by měl mít možnost zadat název měsíce a vrátit počet dní v měsíci.

```python
def pocet_dni_v_mесяci(month):
    # Seznam dní v měsících
    days_in_months = {1: 31, 2: 28, 3: 31, 4: 30, 5: 31, 6: 30, 7: 31, 8: 31, 9: 30, 10: 31, 11: 30, 12: 31}
    
    # Zjištění počtu dní v měsíci
    if month in days_in_months:
        return days_in_months[month]
    else:
        return "Neplatný název měsíce"

# Zvolení programu
month = input("Zadejte název měsíce: ")
print("Počet dní v měsíci:", pocet_dni_v_mēsíci(int(month)))
```

Explanace: Tato funkce definuje slovník s čísly dní v jednotlivých měsících. Používá pak vstupní údaje o názvu měsíce a vrátí počet dní v tomto měsíci. Pokud název měsíce není v seznamu, vrátí chybové poselství.

Řešení 5:
Napište program, který bude řešit úlohu "Kamenný most". Uživatel zadá délku kamenného mostu a rychlost auta, které jej překonává. Program by měl vypsat, zda auto projde mostem a kolikrát se bude muset zbrzdit.

```python
def kamenny_most():
    # Zjištění délky kamenného mostu a rychlosti auta
    délka_mostu = float(input("Zadejte délku kamenného mostu: "))
    rychlost_auta = float(input("Zadejte rychlost auta: "))
    
    # Určení zda auto projde mostem a kolikrát se bude muset zbrzdit
    if délka_mostu / rychlost_auta <= 1:
        print("Auto projde mostem bez zbrzdnutí.")
    else:
        zbrzdnutí = délka_mostu / rychlost_auta - 1
        print("Auto projde mostem a musí se zbrzdit", int(zbrzdnutí), "ráz.")

# Zvolení funkce
kamenny_most()
```

Explanace: Tato funkce definuje proměnné pro délku kamenného mostu a rychlost auta a poté pomocí vstupních údajů určí zda auto projde mostem a kolikrát se bude muset zbrzdit.

Řešení 6:
Vytvořte program, který generuje náhodné čísla od 1 do 100 a následně je seřazení v abecedním rázu. Použijte programovací jazyk Python.

```python
import random

def generování_seřazených_cislo():
    # Generování náhodných čísel od 1 do 100
    náhodná_cisla = [random.randint(1, 100) for _ in range(10)]
    
    # Seřazení čísel abecedním rázu
    seřazená_cisla = sorted(náhodná_cisla)
    
    # Vypis seřazených čísel
    print("Seřazená čísla:", seřazená_cisla)

# Zvolení funkce
generování_seřazených_cislo()
```

Explanace: Tato funkce definuje seznam náhodných čísel od 1 do 100 a poté pomocí funkce `sorted()`
