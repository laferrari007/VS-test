Kvadratickou rovnici x^2 + 5x + 6 = 0 můžeme rozložit na faktory nebo použít kvadratickou formulu:

x = (-b ± √(b^2 - 4ac)) / 2a

V našem případě máme a = 1, b = 5 a c = 6. Přiřaďme tyto hodnoty do kvadratické formuly:

x = (-(5) ± √((5)^2 - 4(1)(6))) / 2(1)
x = (-5 ± √(25 - 24)) / 2
x = (-5 ± √1) / 2
x = (-5 ± 1) / 2

Tímto způsobem lze najít dvě kořeny rovnice:

x1 = (-5 + 1) / 2 = -2
x2 = (-5 - 1) / 2 = -3

protože x1 a x2 jsou řešení rovnice x^2 + 5x + 6 = 0.

Řešení 4:
Úkol 4: Proveďte diferenciální počet funkce f(x) = 3x^2 - 2x + 1.

Funkci f(x) = 3x^2 - 2x + 1 můžeme derivovat pomocí pravidla pro derivaci polynomů. Pravidlo říká, že:

- Derivace konstanty je 0.
- Derivace lineárního členu je jeho koeficient.
- Derivace čtvercového členu je dvojnásobek jeho koeficientu a násobí se jeho kladným exponentem.

Tato pravidla použijeme na funkci f(x) = 3x^2 - 2x + 1:

- Derivace konstanty 1 je 0.
- Derivace lineárního členu -2x je -2.
- Derivace čtvercového členu 3x^2 je dvojnásobek jeho koeficientu 3 a násobí se jeho kladným exponentem 2: 6x.

S kombinací těchto derivací dostaneme derivovanou funkci f'(x):

f'(x) = 6x - 2

Protože derivace funkce f(x) = 3x^2 - 2x + 1 je f'(x) = 6x - 2.

Úkol 5: Najděte integrál následující funkce: ∫(2x^2 + 1) dx.

Integrál ∫(2x^2 + 1) dx můžeme najít pomocí pravidel pro integraci. Pravidlo říká, že:

- Integrál konstanty je konstanta násobená x.
- Integrál lineárního členu je (1/2) koeficient násobený x^2.
- Integrál čtvercového členu je (1/3) koeficient násobený x^3.

Tato pravidla použijeme na funkci 2x^2 + 1:

- Integrál konstanty 1 je 1x = x.
- Integrál lineárního členu 2x^2 je (1/2) jeho koeficient násobený x^2: x^3.
- Integrál konstanty 1 je 1x = x.

S kombinací těchto integrálů dostaneme integrál ∫(2x^2 + 1) dx:

∫(2x^2 + 1) dx = x^3 + x + C

protože integrál funkce 2x^2 + 1 je x^3 + x + C, kde C je konstanta.
