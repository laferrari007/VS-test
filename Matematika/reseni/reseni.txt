Řešení 1: Proveď, že součin dvou množin je prázdná pouze tehdy, jestliže alespoň jedna z množin je prázdná.

Abychom prokázali, že součin dvou množin je prázdná pouze tehdy, jestliže alespoň jedna z množin je prázdná, použijeme následující postup:

Začneme s dvěma množinami A a B. Součin množin A a B je definován jako:

AB = {x | x ∈ A ∧ x ∈ B}

Pokud alespoň jedna z množin je prázdná, pak je součin množin prázdný. Pokud jsou obě množiny neprázdné, pak je součin množin neprázdný.

Zde je důkaz:

1. Pokud A je prázdná, pak není žádný prvek, který by splňoval podmínku x ∈ A ∧ x ∈ B, proto je AB prázdný.
2. Pokud B je prázdná, pak není žádný prvek, který by splňoval podmínku x ∈ A ∧ x ∈ B, proto je AB prázdný.
3. Pokud jsou A i B neprázdné, pak existuje alespoň jeden prvek x, který splňuje podmínku x ∈ A ∧ x ∈ B. Proto je AB neprázdný.

Z těchto tří případů můžeme vidět, že součin dvou množin je prázdný pouze tehdy, jestliže alespoň jedna z množin je prázdná.

Řešení 2: Rozhodněte, zda je rovnoběžnost dvou přímých lineární funkcí určena pouze jejich krokem, nebo zda je nutno přihlédnout i k jejich směru.

Rovnoběžnost dvou přímých lineárních funkcí je určena jejich směrem. Krok funkce není relevantní pro rovnoběžnost.

Dvě přímé lineární funkce f(x) a g(x) jsou rovnoběžné tehdy, pokud jejich-gradienty jsou stejné. To znamená, že pokud f(x) = ax + b a g(x) = cx + d, pak jsou f(x) a g(x) rovnoběžné tehdy, pokud a = c.

Směr funkce je dán hodnotou gradientu, proto je nutné přihlédnout k směru obou funkcí pro určení jejich rovnoběžnosti.

Řešení 3: Najděte všechny řešení systému lineárních rovností:

x + y = 3
2x - 3y = -6

Systém lineárních rovností lze řešit metodou substituce nebo eliminace. Použijeme zde metodu substituce.

1. Přesuneme druhou rovnost o 3y na pravou stranu, aby získali stejný člen y:
2x = -6 + 3y

2. Tímto způsobem můžeme vyjádřit y v první rovnosti:
y = 3 - x

3. Podstatíme-li nyní y v druhé rovnosti, dostaneme:
2x = -6 + 3(3 - x)

4. Rozhodíme-li nyní x, dostaneme:
2x = -6 + 9 - 3x
5x = 3
x = 3/5

5. Pokud nyní vstřebáme x, dostaneme:
y = 3 - 3/5
y = 12/5

Protože máme nyní hodnoty x a y, jsou tyto hodnoty řešení systému lineárních rovností.

Řešení 4: Proveď, že funkce f(x) = 2x^3 - 5x^2 + x - 1 je monotónní na intervalu (0, 3).

Funkce je monotónní, pokud je její derivace kladná nebo záporná v daném intervalu. Derivací funkce f(x) = 2x^3 - 5x^2 + x - 1 je f'(x) = 6x^2 - 10x + 1.

1. Pro určení, zda je f(x) monotónní na intervalu (0, 3), musíme najít kritické body, kde f'(x) = 0.
2. Protože je f'(x) kvadratická funkce, můžeme použít formuli kvadratické rovnosti:
6x^2 - 10x + 1 = 0
3x^2 - 5x + 1/6 = 0

Pro řešení této rovnosti použijeme formuli kvadratické rovnosti:
x = (-b ± √(b^2 - 4ac)) / 2a

V tomto případě je a = 3, b = -5 a c = 1/6.

x = (5 ± √((-5)^2 - 4(3)(1/6))) / (2(3))
x = (5 ± √(25 - 1)) / 6
x = (5 ± √24) / 6
x = (5 ± 2√6) / 6

Protože jsou kritické body x = (5 ± 2√6) / 6, musíme tyto hodnoty porovnat se záporným hranicemi intervalu (0, 3).

Zde je porovnání kritických bodů se záporným hranicí intervalu (0, 3): 
0 < (5 ± 2√6) / 6 < 3

Protože (5 + 2√6) / 6 = 1,5 a (5 - 2√6) / 6 ≈ -0,2, existují dvě kritické body: (5 + 2√6) / 6 a (5 - 2√6) / 6.

3. Protože je funkce f(x) = 2x^3 - 5x^2 + x - 1 na intervalu (0, (5 - 2√6) / 6) neklesající a na intervalu ((5 + 2√6) / 6, 3) rostoucí, je funkce monotónní na intervalu (0, 3).

Protože je funkce monotónní na intervalu (0, 3), je funkce f(x) = 2x^3 - 5x^2 + x - 1 monotónní na intervalu (0, 3).
